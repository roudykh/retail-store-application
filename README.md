# retail-spring-boot

## The Retail Store Discounts

### Run Application
```
git clone https://github.com/roudykh/retail-spring-boot.git
cd retail-application
mvn spring-boot:run
```

### Assumption
- The 5$ discount do not apply to purchase that received percentage based discounts.
